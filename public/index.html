Listed below are the routes that are facilitated by the backend

  • post("/photo")

    Allows logged-in users to post a photo. User will provide the photo, text, and two letter country code.

  • post("/register")

    Allows user to register for access to all the paths.

  • post("/login")

    Allows user to receive JWT for proper authentication

  • get("/timezones")

    Allows user to see list of timezones

  • get("/users")

    Allow users to see list of users

  • get("/photo/:id")

    Allows user to see the photos of a single user

  • get("/photo/timezone/:id")

    allows user to view all photos within in a timezone

  • delete("/photo/:id")

    Allows user to delete photo that belongs to the user
